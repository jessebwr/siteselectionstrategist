# SiteSelectionStrategist

Static Astro site for SiteSelectionStrategist LLC, designed for GitHub Pages.

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

The production build is configured for:

```text
https://jessebwr.github.io/siteselectionstrategist/
```

## Contact Form

GitHub Pages cannot process form submissions by itself. The contact form opens a prefilled email by default.

For real web submissions, create a free hosted form endpoint with Formspree, Basin, Getform, or a similar static-site form service. Then add a GitHub repository variable named:

```text
PUBLIC_CONTACT_FORM_ENDPOINT=https://your-form-endpoint
```
